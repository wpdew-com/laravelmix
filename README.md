# Laravel Mix

Though Laravel Mix was originally built for Laravel projects, it of course may be used for any type of application.

## Usage

### 1: Install

```bash
make install
```

### 2: Watch

```bash
make watch
```

### 3: Build

```bash
make build
```

### 4: Clean

```bash
make clean
```
### 5: help

```bash
make help
```
